C++： 顺序容器

顺序容器概览

容器 头文件 含义
vector vector 动态大小容器，且提供快速随机访问。
deque deque 双端队列实现
list list 双向链表实现
forworld_list forworld_list 单向链表实现
array array 固定大小数组，可以快速随机访问
string string 存储字符串，可以快速随机访问，并且可以在末尾快速插入删除字符

string，vector支持快速随机访问，底层实现是数组；list、forworld_list、list是链表，deque也支持快速随机访问，但是在两端插入和删除数据比list更高效。数组中间插入/删除数据较费时。

array和forworld_liist是新标准中添加的类；forworld_list中没有size函数，其他容器中的size函数是高效的，且有常数时间的复杂度；

defference_type：用于表示两个迭代器的距离；

常用函数

函数 含义
C c(b, e) 通过迭代器初始化c，array类型不可用；
C c{a,b,c} 列表初始化c
C c = {a,b,c} 列表初始化c，array类型不支持
a.swap(b) 交换a和b中的元素；
swap(a,b) 交换a和b中的元素；
c.size() 元素数量(forworld_list不支持)
c.max_size() 最多可持有的元素数量
c.empty() 是否为空
c.insert(args) 插入元素
c.emplace(inits) 使用inits构造c中的元素
c.earse(args) 删除元素
c.clear() 清空元素
==, !=，<，<=，>, >= 比较大小
iterator，const_iterator 迭代器(forworld_list不支持--操作符）
c.begin()，c.end()，c.cbegin()，c.cend() ……
reverse_iterator，const_reverse_iterator 反向迭代器（forworld_list不支持）
c.rbegin()，c.rend()，c.crbegin()，c.crend() ……

仅array，string，vector，deque支持迭代器的算数操作符。其他容器的迭代器不支持。

value_type：容器中含有的元素类型；reference：容器中含有的元素的引用类型；const_reference：容器中含有元素的常数引用类型。

array类型没有默认的构造器。

可以通过向构造器传入迭代器参数来创建容器，原始容器中元素类型可以和新创建的容器类型不同，只要可以执行隐式类型转换；

容器类型的assign函数可以执行容器中元素的隐式类型转换：

swap()：

swap函数交换两个容器中的内容。除了string外，其他类型的容器，指向这个容器中元素的指针、引用和迭代器在交换元素之后会指向另一个容器中的相同元素：

swap()函数有成员函数和非成员函数两种类型，非成员函数在模板中用的比较多；一般推荐使用非成员函数。

关系运算符

所有的容器支持==运算符和!=运算符，只有有序容器支持<，<=，>和>=运算符。只有相同类型的容器才能比较，vector<int>和vector<double也不能；

只有当容器内部元素支持==运算符时，才可以使用容器的==运算符和!=运算符；只有当容器内部的元素支持<运算符时，才可以使用容器的<，<=，>，>=运算符。

插入元素

除了array之外的所有容器支持在末尾插入元素；list、forword_list和deque支持在头部插入元素。

常见的插入元素函数：

函数名称 含义
c.push_back(t)，c.emplace_back(args) 在末尾插入元素t，或是在末尾插入使用args创建的元素，无返回值
c.push_front(t)，c.emplace_front(args) 在头部插入元素t，或者在头部插入用args创建的元素，无返回值
c.insert(p,t)，c.emplace(p, args) 在迭代器p指向的元素前插入t或者根据args创建元素，返回指向添加的元素的迭代器；
c.insert(p,n,t) 在*p前插入n个t元素，返回指向第一个添加的元素的迭代器，如果没有添加元素返回p；
c.insert(p,b,e) 在*p前添加[b, e)中所有元素，返回指向第一个添加的元素的迭代器，如果没有添加元素则返回p。
c.insert(p,il) il是一个用花括号括起来的列表，在p前添加元素，返回指向第一个添加的元素的迭代器，没有添加上元素则返回p。

vector，deque，list和string容器支持insert函数。forword_list只支持在头部插入/删除，vector只支持在尾部插入/删除和随机访问。

访问元素

函数/方法 作用
c.back() 返回列表中最后一个元素的引用
c.front() 返回列表中第一个元素的引用
c[n] 返回列表中第n个元素的引用，不会检查下标是否合法
c.at(n) 返回列表中第n个元素的引用，会检查下标是否合法。如果不合法抛out_of_range异常

需注意的是，容器为空或者下标非法，则以上方法的行为是未定义的。且list和forward_list不支持下标操作符[]和at()函数随机访问元素。

容器如果定义为const，则只能读取容器中的元素。

删除元素

函数 含义
c.pop_back() 删除末尾的元素，无返回值，如果容器为空行为未定义
c.pop_front() 删除开头的元素，无返回值，如果容器为空行为未定义
c.erase(p) 删除迭代器指向的元素，返回指向下一个元素的迭代器
c.erase(b,e) 删除迭代器$[b,e)$范围内的元素
c.clear() 清空容器

forward_list的特殊操作

forward_list将insert，emplace和earse操作用insert_after，emplace_after和earse_after取而代之；

函数 含义
lst.before_begin() 返回指向第一个元素之前的元素的迭代器
lst.cbefore_begin() 返回指向第一个元素的常量迭代器
lst.insert_after(p, t) 在p迭代器指向的元素之后插入元素t
list.insert_after(p, n, t) 在p迭代器指向的元素之后插入n个t元素
lst.inset_after(p, b, e) 在迭代器p指向的元素之后插入迭代器$[b, e)$范围的元素
lst.insert_after(p, il) 在p指向的元素之后插入花括号列表li中的元素
以上操作返回指向最后一个插入的元素的迭代器
emplace_after(p, args) 使用args在p指向的元素之后创建一个新的元素
以上 操作返回新创建的元素的迭代器
lst.earse_after(b,e) 删除迭代器[b,e)表示的范围内的元素，
lst.earse_after(p) 删除迭代器p指向的元素
以上操作返回指向删除元素的下一个元素的迭代器

resize函数

除了array类型的大小是固定的之外，其他容器都可以改变大小。resize函数用来重新设置容器的大小。如果之前容器的大小小了，则在后面添加元素，否则删除后面的元素。

函数 含义
lst.resize(n) 将容器的大小重新设置为n。
lst.resize(n, t) 将容器的大小重新设置为n，新添加的元素的值为t

对容器的操作会使迭代器、引用和指针失效

添加操作：

  • vector和string：在没有重新申请空间的情况下，插入点之前的迭代器、引用和指针不会失效，之后的迭代器、引用和指针都会失效；
  • deque：在除了开头和结尾的任何地方插入元素都会导致迭代器。引用和指针失效，在开头和结尾处插入元素，迭代器会失效，但是引用和指针 不会。
  • list和forward_list：迭代器、引用和指针在插入元素后仍然有效。

删除操作：

  • list和forward_list：迭代器。引用和指针仍然有效；
  • deque：在除了开头和末尾的任何地方删除元素都会导致迭代器、引用和指针失效；在末尾删除元素，end迭代器失效，其他迭代器、引用和指针仍然有效；在开头删除元素，都有效；
  • vector和string在删除点之前的迭代器、引用和指针都有效。

off-the-end迭代器：在插入/删除元素的时候迭代器总是会失效，所以避免保存迭代器的拷贝。

vector和string的：数组大小调整

因为vector和string的内部实现是数组，可以通过如下方法调整数组的大小：

方法 含义
capacity 返回容器所能容纳的元素数量
reserve(n) 调整数组大小，保证容器至少能够容纳n个元素，该操作不会减少数组的容量
shrink_to_fit() 请求将capacity()减少到size()，意即后续操作不会再改变数组的大小。该函数的实现可能会忽略这个请求，deque也实现了这个函数。

string类的特殊函数

构造函数：

函数 含义
string s(cp, n) s是cp所指字符数组前n的字符的拷贝。需要注意cp必须至少含有n个字符
string s(s2, pos2) s是s2中从pos2下标处开始到末尾的拷贝，pos2 > s2.size()后行为未定义
string s(s2, pos2, len) s是s2中从pos2下标处开始到长度len子串的的拷贝，pos2 > s2.size()后行为未定义，拷贝字符数为min(len, s2.size() - pos2)

substr函数：

substr函数用来截取字符串。

函数 含义
s.substr(pos, n) 在s中从pos位置开始截取长度为n的字符串并改回该字符串。
s.substr(pos) 从pos开始截取到末尾的字符串。

当pos超过了string的大小时，substr函数会抛出out_of_range异常。当$pos + n$超过了大小时，截取的字符串为$[pos, s.size())$

string类为insert函数和earse函数提供了传递下标的版本。同样的，stirng类提供了assign函数用来将一个字符数组的元素插入到字符串中。

函数 含义
s.insert(pos,args) 在下标s前插入args，返回指向s的引用。
s.earse(pos, len) 删除从pos下标开始，长度为n的字符串。返回指向s的引用。
s.assign(args) 根据args覆盖s中的内容。返回指向s的引用。
s.append(args) 在s末尾插入args
s.replace(range, args) 将range表示字符串用args代替。range既可以是迭代器，也可以是下标

搜索函数：

s.find()，s.rfind()，s.find_first_of()，s.find_last_of()，s.find_first_not_of()，s.find_last_not_of()。

三种适配器：stack、queue和priority_queue

stack定义在stack头文件中，queue和priority_queue定义在queue头文件中。

成员 含义
size_type 描述容器大小
value_type 元素类型
container_type 适配的容器类型
A a 创建一个适配器，用默认的容器类型；
A a(c) 创建一个适配器，使用容器c的拷贝作为适配；
关系运算 支持关系运算符==，!=，<，<=，>，>=
a.empty() 是否为空
a.size() 元素数量
swap(a,b) 交换a和b中的元素；
a.swap(b) 交换a和b中的元素。

stack：

stack可以作为序列容器的适配器（除了array和forward_list之外）。stack支持的操作有：pop(),push()，emplace()和top()。

queue和priority_queue：

支持的操作有pop()，top()，front()，back()，push()，emplace()。

priority_queue：

需要插入的元素支持<比较操作。
